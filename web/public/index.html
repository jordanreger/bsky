An HTML-only Bluesky frontend.

How to use

Just replace bsky.app with htmlsky.app!

JSON

If you want a raw JSON representation of the profile/post/etc. just add /raw/ before the url.

For example, https://htmlsky.app/raw/profile/jordanreger.com will get you the raw JSON representation of the profile object of jordanreger.com.

Embeds

If you want to use HTMLsky to embed a profile/post/etc. in your own website, just add /embed/ before the url (just like above) and it'll provide a nicer page (i.e. removing the header from the top so the content is on the top).
